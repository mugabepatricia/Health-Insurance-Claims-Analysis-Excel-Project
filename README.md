# Health Insurance Claims Analysis — FY 2024

## Project Overview
An exploratory analysis of 200 health insurance claims records across 5 providers, 3 plan types, and 10 diagnoses. Built as a portfolio project to demonstrate health data analysis skills within a payer analytics context.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Dashboard Design

## Dataset
Simulated claims data containing:
- Member demographics (age, gender, plan type)
- Provider information (name, type)
- Clinical data (ICD-10 codes, diagnoses)
- Financial data (billed, allowed, copay, paid amounts)
- Claim status and prior authorisation flags

## Key Findings
- GERD was the most frequently claimed diagnosis (35 claims), suggesting inadequate outpatient chronic disease management
- Emergency claims carried the highest write-off rate at 51%, consistent with aggressive billing practices in emergency settings
- Prior authorisation was not reducing denial rates — claims requiring prior auth had a 22% denial rate vs 18% for those without, indicating a control gap
- Hypertension had the highest denial rate at 28%, likely driven by medical necessity documentation gaps
- Member M0049 was the highest utiliser at $74,094 billed across 9 claims, flagging a case management opportunity
- January and September were peak claim months, consistent with seasonal respiratory illness patterns

## Dashboard Features
- Interactive slicers for Plan, Gender, Diagnosis, and Service Date
- KPI cards: Total Billed, Total Allowed, Total Paid
- Charts: Billed vs Paid by Provider Type, Claim Status by Diagnosis, Write-off Rate by Provider Type, Claims by Month, Prior Auth vs Claim Status, Top Members by Total Billed
